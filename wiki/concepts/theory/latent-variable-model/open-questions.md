- **Automatic dimensionality determination**: Can $n_z$ be selected automatically, perhaps through a variational approach (as in automatic relevance determination) or information-theoretic criterion (minimum description length)?
- **Beyond pairwise recovery**: Can the pairwise identifiability be strengthened to global identifiability under additional assumptions? What is the weakest assumption needed?
- **Temporal identifiability**: When agents engage in multi-round debate, the latent structure evolves. Can temporal structure (across rounds) provide additional identifiability leverage, analogous to nonlinear ICA with temporal auxiliary variables?
- **Approximate identifiability**: When the sparsity assumption is only approximately satisfied, how gracefully do the guarantees degrade? Is there a quantitative bound on recovery error as a function of Jacobian density?
- **Connection to mechanistic interpretability**: Sparse autoencoders (SAEs) are widely used in mechanistic interpretability to extract monosemantic features from LLM activations. ThoughtComm's sparsity-regularized autoencoder has a similar flavor but regularizes the Jacobian rather than the activations. Could these approaches be unified?
- **Identifiability under model heterogeneity**: The current theory assumes all agents' states are generated by the same function $f$. When agents use different architectures (e.g., LLaMA + Qwen), the generating function differs per agent. Does identifiability still hold under per-agent generating functions with shared latent variables?
- **Scaling with agent count**: Pairwise composition scales as $O(n_a^2)$ — manageable for 3-5 agents but potentially problematic for large swarms. Are there more efficient global recovery algorithms?

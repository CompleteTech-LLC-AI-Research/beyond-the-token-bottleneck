# Latent Variable Model

A **latent variable model** posits that observed data is generated by unobserved (latent) variables through some generating function. The central challenge is **identifiability**: given only the observations, can the latent variables be uniquely recovered? This question has deep roots in statistics, signal processing, and representation learning, and takes on new significance in multi-agent LLM systems where the "observations" are agent hidden states and the "latent variables" are the underlying thoughts driving those states.

In this wiki, the key application is [[thought-communication-multiagent|ThoughtComm]]'s formalization of multi-agent communication as a latent variable model, with three identifiability theorems that provide theoretical guarantees for disentangled thought extraction.
